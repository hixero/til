개발환경 초기화

PowerShell​

PowerShell 설치​

PS> winget install Microsoft.PowerShell -s winget

Oh My Posh​

PS> winget install JanDeDobbeleer.OhMyPosh -s winget

$PROFILE 파일 수정​

PS> notepad $PROFILE

$PROFILE 파일이 없을 경우​

PS> New-Item -Path $PROFILE -Type File -Force

PSSecurityException : UnauthorizedAccess​

PowerShell 현재 사용자 용도​

PS> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

PowerShell 관리자 권한으로 실행​

PS> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine

$PROFILE 파일에 아래 내용 추가​

oh-my-posh init pwsh | Invoke-Expression

Dracula 테마 적용​

위의 파일 수정

oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH/dracula.omp.json" | Invoke-Expression

DirColors​

DirColors 설치​

PowerShell 관리자 권한으로 실행

PS> Install-Module -Name DirColors

DirColors 적용​

Dracula DirColors 파일 생성

PS> New-Item -Path $HOME\.dircolors -Type File -Force
PS> notepad $HOME\.dircolors

프로파일 편집

PS> notepad $PROFILE

테마 적용

Import-Module DirColors
Update-DirColors $HOME\.dircolors

git​

git 설치​

PS> winget install --id Git.Git -e --source winget

git bash 테마 적용​

git bash> vi .bashrc
eval "$(oh-my-posh init bash --config $HOME/AppData/Local/Programs/oh-my-posh/themes/dracula.omp.json)"
