Docusaurus

Prequisites​

개발환경​

Node.js​

Install​

Docusaurus 프로젝트 생성​

PS> npx create-docusaurus@latest til classic --typescript
PS> yarn add @docusaurus/theme-live-codeblock
PS> yarn add @docusaurus/theme-mermaid

Run​

Docusaurus 실행​

PS> yarn run start

Configuration​

테마설정​

docusaurus.config.ts
const config: Config = {
  themes: ['@docusaurus/theme-live-codeblock', '@docusaurus/theme-mermaid'],
  markdown: {
    mermaid: true,
  },
  themeConfig: {
    ...defaultThemeConfig,
    colorMode: {
      defaultMode: 'dark',
      disableSwitch: false,
      respectPrefersColorScheme: false,
    }
  },
  prism: {
    ...defaultPrismTheme,
    additionalLanguages: ['powershell','json','typescript','python','bash'],
  }
}
